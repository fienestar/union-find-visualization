# [union-find-visualization](http://union-find.kro.kr)
union-find 동아리 keeper 발표 자료

2021.12.02 약 4시간

명령어
- sample
  - 모든 노드를 제거하고 0, 1, ..., 9 노드를 생성합니다.
- reset
  - 모든 노드를 제거합니다.
- merge A B
  - A와 B를 병합합니다.
  - union by rank 등은 적용되지 않으며 A에 B를 병합합니다.
- create A:
  - A노드를 생성합니다.
- parent A:
  - A가 속한 트리의 루트노드를 구하여 출력합니다.
